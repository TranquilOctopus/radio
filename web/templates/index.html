iPhone alarm sync setup

Create an iOS Shortcut so your wake-up alarm syncs automatically:

  1. Open the Shortcuts app → Automation tab
  2. Tap + → New Automation
  3. Scroll to Alarm → choose When my Wake Up Alarm changes
  4. Add action: URL → enter http://splitflapclock.local/alarm
  5. Add action: Get Contents of URL
  6. Set Method: POST, Headers: Content-Type: application/json
  7. Body (JSON): {"time": "HH:MM"} — replace HH:MM with the Alarm Time shortcut variable
  8. Save and enable the automation

The clock is reachable at http://splitflapclock.local on your local network.
